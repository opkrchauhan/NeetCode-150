<h2><a href="https://www.geeksforgeeks.org/problems/encode-and-decode-strings/1">Encode and Decode Strings</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18.6667px;">Given an array of strings <strong>arr[]</strong>, implement an algorithm in the <strong>encode()</strong> function that can convert the given strings into a <strong>single encoded</strong> string, which can be transmitted over the network and then <strong>decoded</strong> back into the <strong>original array</strong> of strings from the encoded string. The decoding will happen in the <strong>decode()</strong> function.</span></p>
<p><span style="font-size: 14pt;">The encoding rule used here is:</span></p>
<ul>
<li><span style="font-size: 14pt;">Each string is stored as length + "/:" + actual string.</span></li>
<li><span style="font-size: 14pt;">During decoding, we first read the length before the delimiter "/:", and then extract that many characters as the original string.</span></li>
</ul>
<p><span style="font-size: 14pt;">This approach works for all ASCII characters since the length ensures correct parsing regardless of string content.</span></p>
<p><span style="font-size: 14pt;"><strong>Note:</strong> You are not allowed to use any inbuilt serialize method.&nbsp;&nbsp;</span></p>
<p><span style="font-size: 14pt;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr[] =<strong> </strong>["Hello", "World"]<strong><br>Output:&nbsp;</strong>["Hello","World"]<strong><br>Explanation: <br></strong>The encode() function converts the array into a single string by storing each word as length + "/:" + string.
For the given input, "Hello" becomes "5/:Hello", "World" becomes "5/:World". These are concatenated to form "5/:Hello5/:World", which is transmitted.<br>
During decoding, we read the length before "/:", then extract that many characters:
Read 5 then extract "Hello"
Read 5 then extract "World"
Thus, the original array ["Hello", "World"] is reconstructed.</span></pre>
<pre><span style="font-size: 14pt;"><strong>Input: </strong>arr[] =<strong style="font-size: 14pt;">&nbsp;</strong><span style="font-size: 14pt;">["abc", "!@"]</span><strong><br></strong><strong>Output:&nbsp;</strong>["abc","!@"]<strong><br>Explanation:&nbsp;</strong>The encode() function converts each string into length + "/:" + string and concatenates them. Here: "abc" becomes "3/:abc", "!@" becomes "2/:!@" then encoded string = "3/:abc2/:!@".
During decoding, lengths are read to extract exact substrings, reconstructing ["abc","!@"].<br></span></pre>
